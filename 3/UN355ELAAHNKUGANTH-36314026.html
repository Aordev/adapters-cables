Description:
The item is a  2 in 1 USB 3.1 Type C to HDMI+VGA Female Adapter for Macbook Laptop Google New Chromebook Pixel.
It is mainly used for thinner more delicate device video output.
Its biggest feature is to support the VGA and HDMI display at the same time, on different monitors.

Features:
1. Ultra-thin design, mini size
2. Aluminium alloy housing, high-end fashion design
3. The output of interfaces: external HD VGA, HDMI equipment such as: Monitor Projector with HD television equipment
4. Used in Apple notebook Macbook Air 12-inch USB3.1 HD VGA output terminal for external devices.

Specification:
1. Color: Gold, Silver
2. The shell material: Aluminium alloy 
3.  Interface: USB3.1 Type-C Male to HDMI+VGA Female
4. HDMI Output supports 4K resolution, VGA supports 1080P.
5. Compatible with: Macbook12'', Macbook Pro13'' and 15'', Google New Chromebook Pixel, and other laptop which has a Type C port.
6. The product size: Approx. 58*39*15 mm
7. The package size: Approx. 140*72*17 mm
8. The cable length: Approx. 6cm
9. The weight: Approx. 55 g

Important Notes:
- The adapter works for Thunderbolt 3 compatible Type-C port only.
- Directly connect the adapter to the device (not another adapter).
- This adapter is built with the solution of (DP ALT MODE), please make sure your devices also support DP ALT MODE.
(DP ALT MODE RESOLUTION, It NOT  work with USB-C (Type C) SmartPhone ! (USB-C Smart Phone work mostly with MHL resolution)!

Warm reminding:
1.  It needs 8 to 10 seconds for the data to synchronize from computer with your monitor.                
2.  When the signal can not be transmitted, please check whether the interfaces on both ends are tightened,or try to make usb-c interface flip and then insert.
