Description 
Ugreen USB to Parallel Printer Adapter Cable is ideal for connecting USB-equipped desktop or notebook computers to a parallel printer, injekt, laser, and scanner etc, or for adding a DB25 female port. It is compatible with USB 1.1 and 2.0, plug & play, no driver installation is required. 

Specification 
1. Connector: USB 1.1 type A male x1; Parallel Sub-D 25 pin female x1 
2. Support data transfer speed up to 12Mbps 
3. Parallel DB25 female with screw 
4. provide a full duplex, bidirectional communications bus and can be used for any IEEE-1284 application 
5. Plug and Play Installation
