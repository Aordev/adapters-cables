รายละเอียดของสินค้า 3in1 USB 3.1 Type C to HDMI+VGA+3.5mm Audio Female Adapter for Macbook 12 I - intl

3 in 1 USB 3.1 Type C to HDMI+VGA+3.5mm Audio Female Adapter for Macbook 12 Inch Laptop Google New Chromebook Pixel
 
Description:
The item is a 3 in 1 USB 3.1 Type C Male to HDMI+VGA+3.5mm Audio Female Adapter.

Product Usage:
Mainly for more delicate device video output for MSI board computer, Apple MacBook Type-c, Hewlett-Packard, Google computer.

Specification:
Color: Gold
Material: Aluminium alloy body
Version: 3.1, support resolution up to 1080P.
Input: USB-C/USB 3.1 TYPE-C
Output: HDMI, VGA, 3.5 audio interface
Resolution: HDMI VGA output support maximum resolution: 1080P@60Hz
The body size(not included cable): Approx. 78*51*15 mm /3.07*2*0.59 inch
The cable length: Approx. 100 mm / 3.94 inch
The weight: Approx. 60 g
 
Type-C Compatibility List ( List Reference model)
- 2016 MacBook Pro
- 2015 MacBook 12"
- ChromeBook Pixel
- Dell XPS 13-9350-R1609 13.3", XPS 15
- Acer V15 Nitro
- ASUS GA-Z170X
- Samsung Notebook 9 900X5L-K01
- HuaWei MateBook
- Acer Switch Alpha 12
- hp SpecterX2
- Cube i7 book

- ASUS Zenbook 3

 

Important Notes:
- The adapter works for Thunderbolt 3 compatible Type-C port only.
- Directly connect the adapter to the device (not another adapter).
- This adapter is built with the solution of (DP ALT MODE), please make sure your devices also support DP ALT MODE.
(DP ALT MODE RESOLUTION, It NOT  work with USB-C (Type C) SmartPhone or USB-C Tablet! (USB-C Smart Phone/Tablet work mostly with MHL resolution)!

 

Warm reminding:
1.  It needs 8 to 10 seconds for the data to synchronize from computer with your monitor.                
2.  When the signal can not be transmitted, please check whether the interfaces on both ends are tightened,or try to make usb-c interface flip and then insert.
