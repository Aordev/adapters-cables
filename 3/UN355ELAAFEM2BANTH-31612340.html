รายละเอียดของสินค้า สาย USB Am to mini USB 5pin V2.0 ยาว 3M (สีดำ)

USB 2.0 Type A ขนาดตัวผู้ 5 ตัวยึดสายเคเบิลเชื่อมต่อคุณลักษณะ Type A USB มาตรฐานอยู่ทางปลายแถวด้านหนึ่งคนตัวเล็ก Type b ต่อไปอีก คุณสามารถเสียบ type a เชื่อมต่อเข้าสู่คอมพิวเตอร์ของคุณ แล้วเสียบมินิ type b เป็นอุปกรณ์เชื่อมต่อกับมินิ type b พอร์ต อุปกรณ์อย่างโทรศัพท์มือถือ MP3 เล่น ยา กล้องดิจิตอล และอุปกรณ์กล้องดิจิตอลขนาดเล็กมักมี type b พอร์ต

ข้อกำหนด

1 ตัวที่ 1: USB 2.0 Type B ชาย ต่อ 2: มินิ USB 2.0 5 เข็มกลัดตัวผู้

2 USB 2.0 ความเร็วสูงเพื่อมาตรฐานอัตรา 480 MB s/

3 ใช้งานร่วมกับเครื่องพีซี และ mac

4 สนับสนุนแบบ Plug and play

5 การกัดกร่อนทนตัวนำทองแดงเปลือย ชุบทองตัว และทำลาย& เอาให้สุดเกลียวไฟฟ้า และลดการสูญเสียข้อมูล

ความกว้าง :

Mini usb 2.0 รองรับสายสำหรับกล้อง GoPro HERO, HERO 3+จอย PS3, Astro a40/50, MP3 เล่น HTC กล้องดิจิตอล canon กับ
