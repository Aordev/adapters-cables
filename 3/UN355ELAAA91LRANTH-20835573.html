รายละเอียดของสินค้า 1x 3 in 1 Mini Display Port Thunderbolt to DVI VGA HDMI TV AV HDTV Adapter Cable for MacBook iMac Air Mac Book Pro(White) - intl

Specification:
Output Video: HDMI/DVI/VGA
Input: Mini Display Port Male 20pin
Output: HDMI Female Type A 19pin
Connector:DVI Female (24+1) VGA Female 15Pin
Supports max resolution for HDMI port: 4K x 2K
Mini DP Port support DP v1.2, which is for 1.62 Gbps, 2.7 Gbps, 5.4 Gbps link rates
HDMI Port support v1.4, and the resolution of 4Kx2K, 1080p, 720p, and HDMI Audio
DVI Port support v1.0, and the resolution of 1080p, 720p
VGA Port support max resolution of 1920x1200, and others like 1920x1080, 1024x768

Note:
- Can only convert signal from Mini DisplayPort to HDMI/DVI/VGA. This is not a bi-directional adapter
- It does not support more than one output simultaneously
- No audio output from VGA and DVI. Macs made before 2010 do not output audio over Mini DisplayPort
- Please make sure the movie is 4K x 2K and make sure TV or Display support 4K x 2K resolution, then you can get the 4K x 2K image, or you only get standard image(not 4K result)

Compatibility (Incomplete List)
- Apple MacBook, MacBook Pro, MacBook Air, iMac, Mac mini, Mac Pro
- Microsoft Surface Pro/Pro 2/Pro 3 for Windows 8 Pro
- Lenovo ThinkPad X1 Carbon
