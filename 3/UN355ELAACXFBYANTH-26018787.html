รายละเอียดของสินค้า ช่องต่อ USB 2.0 แบบ 7 ช่อง 7 Port High Speed USB 2.0 Hub (สีขาว)

  ช่องต่อ USB 2.0 แบบ 7 ช่อง 7 Port High Speed USB 2.0 Hub ที่มาพร้อมกับเทคโนโลยีการเชื่อมต่อความเร็วสูงแบบ USB 2.0 ซึ่งมีความเร็วสูงสุดถึง 480Mbps ทำให้สามารถโอนถ่ายข้อมูลได้อย่างรวดเร็วแม่นยำ และมีประสิทธิภาพ หมดปัญหากับช่องต่อ USB ที่ไม่เพียงพออีกต่อไป..ห้ามพลาด!!

# คุณสมบัติ

- อัตราการรับส่งข้อมูลสูงสุด 480Mbps
- สามารถใช้งานร่วมกับ USB1.1 หรือรุ่นที่ต่ำกว่าได้
- พอร์ต USB จำนวน 7 พอร์ต สำหรับการขยาย และติดตั้งอุปกรณ์เพิ่มเติม ใช้งานง่าย
- สนับสนุนการใช้งานแบบ Plug-and-play
- มีช่องเสียบ อะแดปเตอร์จ่ายไฟภายนอก
- สำหรับแต่ละช่องต่อ USB2.0 เชื่อมต่อได้สูงสุด 500mA
- ความยาวของสายเคเบิล 0.6 ม.
- ขนาดกะทัดรัด และน้ำหนักเบา ง่ายต่อการใช้งาน

# ขนาด 165 x 34 x 15 มม.
