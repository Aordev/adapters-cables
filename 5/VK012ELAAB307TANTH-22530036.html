Supports 4 downstream ports for all speeds.
Low-speed (1.5Mb/s) / Full-speed (12Mb/s) / High-speed (480Mb/s) / Super-speed (5Gb/s).
Single-lane (x1) PCI Express throughput rates up to 5 Gb/s.
Backwards compatible with USB 2.0 devices.
Supports simultaneous operation of multiple USB 3.0 and USB 2.0 devices.
Molex power required.
Hot-swapping feature allows you to connect/disconnect devices without powering down the system.
Support UASP: 70% Faster than traditional USB 3.0, with UASP Transfer Protocol; The UASP Transfer Protocol only can accelerate read and write speed of SSD hard disk.

Specification:
Chipset: Renesas uPD720201
Internal Ports: 1x SATA 15Pin Power Connector
Host Interface: PCI Express (PCI-E)
Data Transfer Rate: Up to 5.0 Gbps
Output: 5V/1A MAX (each USB port)
Color: Black, Red
Operating Systems:
Windows XP, Vista and Windows 7 (with included drivers)
Windows 8/8.1/10 - No additional drivers are needed for Win8/8.1/10
Windows XP and Vista Users Please Note:Due to OS limitations, the real transfer speed after installation of the respective drivers will be located between the level of USB 2.0 and 3.0
