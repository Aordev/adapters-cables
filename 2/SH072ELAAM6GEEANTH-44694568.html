OEM USB 2.0 Ethernet Adapter สำหรับต่อระบบ LAN ผ่านทาง USB Port รุ่น CC-048A

เป็นอุปกรณ์แปลง Port USB เป็น Port สำหรับต่อเข้ากับระบบ LAN ติดตั้งง่าย เพียงเสียบอุปกรณ์เข้าไปทาง Port USB ติดตั้ง Driver ก็สามารถใช้งานได้เหมือนใส่การ์ด LAN เพิ่มอีก 1ตัว เหมาะสำหรับคอมพิวเตอร์ที่ไม่มีการ์ดแลน คุณสมบัติเทียบเท่าการ์ดแลน PCI 10/100 Mbpsรองรับ Windows XP และ Linux รองรับ Windows7

Specification:
- USB to Ethernet Lan RJ45 Network card.
- Enable to connect to the internet via the USB port.
- RJ45 female to USB male.
- Compliant to USB interface version 1.0/1.1/2.0.
- Compliant with IEEE 802.3 (10Based-T) and 802.3u (100Based-TX) standards.
- Support both full-duplex and half-duplex operations.
- Support suspend mode and remote wakeup via link-up and magic packet.
- Support for Windows 98ME/XP/Vista, Mac 10.X or above, Linux.
- Powered by USB port, no external power adapter required.
- LED indicator.
