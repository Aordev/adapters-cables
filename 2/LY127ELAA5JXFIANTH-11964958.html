รายละเอียดของสินค้า LYBALL type C ถึง 4กิโลไบต์จท HDMI USB 3.0 HUB USB-C อะแดปเตอร์พอร์ตรุก UBS 3.1 สาย 3ใน1 รายละเอียด: สี: เงิน USB 3.1 (รุ่น 2) type C เฟสขึ้นอยู่กับอัตราการถ่ายโอนข้อมูล 10 Gbps ใช้งานง่าย: Aand เล่นปลั๊กไฟ ไม่จำเป็นต้องมีอีกคน การ USB-C multiport กล่องโลหะอลูมิเนียมอะ แด็ปเตอร์ทั้งหมด เพื่อสนับสนุนการแก้ปัญหาเตียงฟูลช่อง 4 KB/2160p (4 KB) ที่มีความละเอียดสูง ความยาวสายเคเบิล: ประมาณ 3.6นิ้ว (91.5 มิลลิเมตร) ขนาดกล่อง: ประมาณ 2.4นิ้ว x 2.0นิ้ว x 0.5นิ้ว (60.0 มิลลิเมตร x 50.0 มิลลิเมตร x 11.5 มิลลิเมตร) สามารถเชื่อมต่อดิสก์ ฮาร์ดดิสก์ เมาส์ คีย์บอร์ด โทรศัพท์ กับ Mac OS ระบบปฏิบัติการ Chrome ใน และ Windows 10 3 พอร์ตเอาต์พุต: USB 3.0 type A มาตรฐานส่งออกสตรีหมายเลข HDMI พอร์ตเอาต์พุตหญิง รองรับ 1080P USB-C หญิงเก็บพอร์ต (พุ่งเข้าพอร์ตไม่รองรับการถ่ายโอนข้อมูล สาย AV ออก ส่งออกรุก และขยายตัว) รวมบรรจุภัณฑ์: 1ชิ้น x เงินอะ แด็ปเตอร์ฮับ USB 3.1 Type C HDMI หมายเหตุ: ใช้งานได้เฉพาะกับ macbook! โปรดให้ข้อ 1 และ 2 เซนติเมตรวัดจากมือ สีภาพถ่ายสินค้าอาจแสดงในการแสดงต่าง ๆ บนหน้าจอคอมพิวเตอร์จอภาพของคุณปรับ เพราะไม่ถูกกัน
